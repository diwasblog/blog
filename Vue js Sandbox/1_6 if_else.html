Dynamically binds an attribute to an expression.Some html attribute are : id,src,title,style,width,height, disabled etc
Can be Used as

v-bind:id

 v-bind:src="myimage"

  v-bind:style="color:red",

v-bind:href  
 Eg :<item optional="1">...</item> Here, place of optional is Attribute Name(Attribute) and "1" is attribute value(Expression)

Example

In Stock

Almost Solid

Out of Stock

<html> <head> <title>If Else</title> </head> <body> <style> .active { background: red; } </style> <div id='app'> <p v-if=" inventory > 10 ">In Stock</p> <p v-else-if=" inventory < 10 && inventory > 0">Almost Solid</p> <p v-else>Out of Stock</p> </div>
<script> const app=new Vue({ el:'#app', data:{ inventory:0 } }); </script> </body> </html>
