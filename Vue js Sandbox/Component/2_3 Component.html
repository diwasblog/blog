Component is reusable Vue instance.They accept the same options as new Vue, such as data, computed, watch, methods, and lifecycle hooks. The only exceptions are a few root-specific options like el.

Data Must be Function

 You may have noticed that data wasn’t directly provided an object, like this:

Instead, a component’s data option must be a function, so that each instance can maintain an independent copy of the returned data object:

If Vue didn’t have this rule, clicking on one button would affect the data of all other instances, like below:

Example
