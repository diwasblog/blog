1. Component are reusable Vue instances with a name.

  2. Since components are reusable Vue instances,they accept same options as new Vue such as data, computed, methods and watch and lifecycle hook.

  3. Only component donot have is el element

1 Use of Component

<div id="components-demo"> <button-counter></button-counter> </div> <script> Vue.component('button-counter', { data: function () { return { count: 0 } }, template: '<button @click="count += 1">You clicked me {{ count }} times.</button>' }) new Vue({ el: '#components-demo' }) </script>

Reusable Component

Notice that when clicking on the buttons, each one maintains its own, separate count.

That’s because each time you use a component, a new instance of it is created.

<div id="components-demo2" > <button-counter></button-counter> <!--using component < button-counter > =>new instance of its is created. so separate copy of count for this buttoncounter--> <button-counter></button-counter> <!--using component < button-counter > =>new instance of its is created. so separate copy of count for this buttoncounter--> <button-counter></button-counter> <!--using component < button-counter > =>new instance of its is created. so separate copy of count for this buttoncounter--> </div> <script> Vue.component('button-counter', { data: function () { return { count: 0 } }, template: '<button @click="count += 1">You clicked me {{ count }} times.</button>' }) new Vue({ el: '#components-demo2' }) </script>
