LifeCycle Hook vue  in serial order

  • Created
  • Mounted
  • Updated
  • Destroyed

First, the component is created. Here, everything — including reactive data, computed props, and watchers — are set up and initialized.

Second, the component is mounted to the DOM. This involves creating the actual DOM nodes and inserting your component into the page.

Third, your component is updated as reactive data changes. Vue will re-render your component many times, in order to keep everything on the page up-to-date.

Lastly, when the component is no longer needed, it is destroyed.Any event listeners are cleaned up, DOM nodes are removed from the page, and any memory it was using is now released.
