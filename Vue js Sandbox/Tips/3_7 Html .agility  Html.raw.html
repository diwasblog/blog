Html Raw

Text output will generally be HTML encoded. Using Html.Raw allows you to output text containing html elements to the client, and have them still be rendered as such. Should be used with caution, as it exposes you to cross site scripting vulnerabilities. @Html.Raw("<h2>Hello world</h2>"); OUtput:

Hello world
