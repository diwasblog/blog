One Way Data Binding Two Way Data Binding
Data flows only from object to UI and not vice versa Data Flow in both direction becoz data schronization occur here. We use v-model for this.
The UI binded with the object having property;when the object value changes .It will be reflected on UI, no update in object property. Here , any change in object is reflected to UI and any change in UI is reflected in object.

Two way binding:
