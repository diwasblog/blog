Accept Alphabet, Space and backspace

  
    <input type="text" onkeydown="onkeydownAlphabetValidate()" class="required form-control" id="LocalLevelEmployeeName" name="LocalLevelEmployeeName" />
  
    function onkeydownAlphabetValidate() {
      var ch = String.fromCharCode(event.keyCode);
      var filter = /[a-zA-Z\s\b]/;
    
      if (!filter.test(ch)) {
          event.returnValue = false;
      }
  }

Accept only Alphabet but not _ and other symbol

  
    <input type="text" id="uname" name="uname" pattern="[a-zA-Z0-9]+" minlength="4" maxlength="10">

    Valid: monty42, 42monty, MON42ty,  mon42ty
    Invalid:  monty_42

Always start with _

  
    <input type="text" id="uname" name="uname" pattern="_[a-zA-Z0-9]+" minlength="4" maxlength="10">

Email always ends with tutsplus.com , envato.com

  
    <input type="email" id="email" pattern=".+@tutsplus\.com|.+@envato\.com">
