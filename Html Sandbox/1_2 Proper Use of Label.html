Use of label For

'for' of the label must match with the id of input field so that when you click on label the input field will be highlighted. 
              
                <form>
                    <label for="FN">First Name</label>
                    <input type="text" name="lname" id="FN" disabled><br>
                    <input type="submit" value="Submit">
                </form>


==============================================================================================
Without label for (click on label textbox willnot be highlighted.) 
            
              <form>
                  <label>First Name</label>
                  <input type="text" name="lname" id="FN" disabled><br>
                  <input type="submit" value="Submit">
