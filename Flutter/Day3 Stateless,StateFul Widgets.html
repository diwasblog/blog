A widget is either stateful or stateless. If a widget can change—when a user interacts with it, for example—it’s stateful.

Stateless Widgets StateFul Widgets
The state of the Widgets doesn't change over time. The state of the widgets changes over time.
  • Icon, IconButton, and Text are examples of stateless widgets.
  • Stateless widgets subclass StatelessWidget.
  • Checkbox, Radio, Slider, InkWell, Form, and TextField are examples of stateful widgets.
  • They change with user input.
  • Stateful widgets subclass StatefulWidget.

                    import 'package:flutter/material.dart';

                    void main() => runApp(MaterialApp(
                      home:Home(),  // Using Own Home Widget 
                    ));
                    
                    class Home extends StatelessWidget{
                      @override
                      Widget build(BuildContext context) {
