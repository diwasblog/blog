Flutter 

  • UI toolkit for building fast, beautiful, natively compiled applications for mobile, web, and desktop with one programing language called dart.
  • Initially developed by Google and now manages by ECMA Standard.

Flutter Engine

Flutter Engine is written primarily in C++. It interfaces with platform-specific SDKs such as those provided by Android and IOS.

Widgets 

Widgets are nothing but the class which includes all the text, shapes, and animations. So, used for creating graphics.

Ahead of Time Compiler Just In Time Compiler    
Compilation takes place prior to running. Compilation produces a native executable machine code.
  • Compilation takes place as the program is running.
  • The source code is converted on the fly to the machine code.
  • Compilation never take place.
  • Interpreter interprets source code and convert to desired result.
  • No conversion to machine code occurs.
  • Compilation to bytecode occurs prior to running.
  • compilation produces a bytecode designed to be interpreted.
  • Then bytecode gets converted to machine code at runtime.

Downloading Steps

  • Android Studio: https://developer.android.com/studio
  • Git: https://git-scm.com/downloads
  • flutter SDK: https://flutter.dev/docs/get-started/install      ====> download using git clone to get latest flutter sdk
  •  open flutter console 
  •  flutter --version                ====> It may install dart sdk at this time

Then your must access flutter --version from every where.

Steps:

  • Search: env
  • Goto Environment Variable button
  • Click on Path
  • Add New path: Here put bin path of flutter installed. In my case :  C:\src\flutter\flutter\bin
  • Press Ok

Now, you can get flutter from anywhere on your system.
