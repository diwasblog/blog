Beam, Slap, Shear Walls, Column

Beam   Lintel
The beam carries the load from the slab/floor/roof and transfers it to the column. From column, it is transferred to the footing, and from footing finally to the soil.   Lintel carries the load of the wall above the openings/door/window, and transfer it to the walls only.
It’s end point is rested on columns   It’s end point is rested on walls
It does not contain door and windows under it   It contains door and windows under it

RCC vs PCC

PCC RCC
Plain Concrete Cement Reinforced Concrete Cement
It doesnot contains Steel It contains Steel.
It is strong in compression and weak in tension It is strong in compression and tension.
So, it cannot be used for column, beam, slab which has to be bear load and tension. It can be used in column, beam and slab.
