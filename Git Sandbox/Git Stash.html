Suppose you are working on some branch and you haven't completed the work so commiting  impartial work is bad practice so put this incomplete task in stash and do other task and when that task is finished then return back to stashed code. 

Note:Stash work as stack basis 

  1)Add diwas.txt file
  2)In Git :
       git add .
  3)Donot commit because we haven't complete the task on this file 
  4)In Git :
     git stash 
  5)Do other work
  6)In Git: 
      git stash list
  7)In Git :
      git stash pop
  When you pop stash then that stash record willbe deleted
