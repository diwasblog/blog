General Tech Terminology

Client / Server

  • Client requests for the resource to the server.
  • The server responds to the resource to the client.

Google server, and we are a client requesting google server for resources and resources can be anything like image, video, text, etc.

API (Application Program Interface)

  • It is for software to software interaction.
  • Suppose I am using a web browser and uses facebook then I am not using API directly, the facebook may be using numerous API for getting data, image, location, from database etc.
  • Todays we use it differently, facebook uses google api for logining with Gmail. Then facebook is using google api So we using API from google company services like google map and building our own services.
  • So we using API from google company services like google map and building our own services. Sometimes we need to pay for api

SDK (Software Development Kit)

  • A large collection of API. 
  • For making an android application, we need Java SDK.
