Logical Core vs Physical Core

CPU in a CPU socket may have many physical cores.

If a CPU processor has 2 physical core then it is called a dual-core.

If a CPU processor has 4 physical core then it is called 4 core.

 

Logical Cores = # of Physical Core *  Thread on each core   

 If I have a computer that has a 4-core processor, runs two threads per core, then 

Logical Processor = 4 * 2 = 8 Logical Processor,

This concept of behaving 1  processor (physical processor) as 2 processor (logical processor) is hyperthreading. Hyperthreading makes a Physical Processor to behave like it has two Physical Processors, which are called Logical Processor.

This above hardware has a processor with 4 core (obtained called as 4 physical processor units) and has 2 thread in each core so altogether this processor has (4*2) =8 logical processor(as shown right of the figure).

Each physical core has its own circuitry and its own L1 (and usually L2) cache can read and execute instructions separately (for the most part) from the other physical cores on the chip.

 

A quad-(physical) core i7 processor will have eight logical cores. However, the two logical cores within one physical core cannot truly operate in parallel with respect to each other. But works interleaving or concurrency so that one logical processor does half of the task on one work and another logical processor does half on other work and soon and continuously interleaved.
