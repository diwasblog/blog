Visual Studio Support
.NET Core 2.2 does have support inside Visual Studio but only from version 15.9 Preview 3 onwards. So update your visual studio.
 
 
.Net Core 
.Net core has two parts.
 
1) SDK 
2) runtime
 
SDK
Always download the latest SDK whatever .net core project you have.
 
Suppose you have .net core application build using 2.1 version runtime then when you give that code source to your friends then he/she also needs the same version of .net core runtime. He can have any version of .net core SDK.
 
The software development kit (SDK) includes everything you need to build and run .NET Core applications, using command line tools and any editor (including Visual Studio).
 
To develop, build and publish an application will require an SDK.
 
 
runtime
runtime is required to run .net core application. 
 
The runtime includes just the resources required to run existing .NET Core applications. The runtime is included in the SDK.
 
 

Runtime: to run apps (Developer need this)

SDK (Runtime + Tooling): to build and run apps (Developer need this)

AspNetCoreModule :Actually server need to install this to run .net core application

 
Check SDK and runtime in your system
 
1) Open cmd and type dotnet --info
you will get both runtime and sdk.
 
From sdk 2.x onward , it will automatically install same runtime. i.e Installing an SDK also installs the runtime.
