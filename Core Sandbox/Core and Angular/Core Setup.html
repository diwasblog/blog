Step 5. Create ASP.NET Core Web API Project using dotnet new command

    
mkdir DatingApp                         ===> Make a Webapi Project Folder
cd DatingApp                             ===> Move inside folder
dotnet new webapi -h DatingApp.API       ===> Create web api project name: DatingApp.API
cd DatingApp.API                         ===> Navigate to DatingApp.API 
dir                                       ===> To see the content
code .                                    ===> To open DatingApp.API in vscode
