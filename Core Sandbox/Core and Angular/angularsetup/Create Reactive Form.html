Types of Angular Form

1 ) Template Driven Form   

2 ) Reactive Form

Template Driven Form Reactive Form / Model Driven Form
Make Use of "FormModule" Make Use of "ReactiveFormsModule "
Asynchronous in nature  Synchronous in nature
Easy to build and understand More scalable, reusable, and testable 
Template-driven form can be used for basic form requirements such as login form. It is not recommended to create complex forms using template-driven form because the HTML part will be complex and messy. Used for complex forms with more fields.
the template-driven form is created using directive Reactive forms are created in the component class 
template-driven forms are validated using directives Reactive forms can be validated using functions 
Difficult to do the unit test because most of the code is written in an HTML file Easy to unit test because most of the code is written in the component ts file.
We completely build a form in HTML. We completely build a form in code.
  Only Downside: It required more code then template-driven form.

 

Building Block of Model Driven Form


How to use Reactive Forms

  1. Import ReactiveFormsModule
  2. Create Form Model in component class using Form Group, Form Control & Form Arrays
  3. Create the HTML Form resembling the Form Model.
  4. Bind the HTML Form to the Form Model

            
                import { ReactiveFormsModule } from '@angular/forms';

                @NgModule({
                      imports: [
                            ------
                            ReactiveFormsModule
                      ],
                      ------
                })
                export class AppModule { }

Using FormBuilder

Same as Previous , No need to make changes on .html

It is used to create a big reactive form with minimum code in Angular.

FormBuilder has methods as group(), control() and array() that returns FormGroup, FormControl and FormArray respectively. 

            
                import { FormGroup, FormControl, FormBuilder } from '@angular/forms';

                export class EmployeeDocumentComponent implements OnInit {
                
                
                  employeeDocuments: FormGroup;
                  constructor(private fb: FormBuilder) { }
                
                  ngOnInit() {
                     this.employeeForm = this.fb.group({
                                    fullName: ['',Validate.Required],
                                    email: [''],
                                  });
                
                
                  }
                
                }
