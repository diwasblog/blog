constructor ngOnInit
Concept of  It is the concept of Typescript and Javascirpt. It is called by Javascript Library. It is the concept of Angular. It is called by Angular framework
Binding State Binding has not happened till this moment so only class variables are accessible. Binding with UI has been done so the class variable has a value set from UI
DOM The component is not initialized. So, DOM is not accessible. The component is initialized, DOM is ready and accessible. If you are doing any manipulation of the component's DOM via e.g. ViewChildren, ContentChildren or ElementRef, your native elements will be available during the this phase.
Initialization Logic Initialization of the class variable and dependency injection is done here. Mainly done for the child-parent data passing concept here.
