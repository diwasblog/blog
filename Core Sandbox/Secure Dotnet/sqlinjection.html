SQL Injection Attack

SQL Injection attack is one of the most dangerous attacks it is ranked 1 in the top 10 Vulnerabilitiesby OWASP2013 [Open Web Application Security Project]. SQL injection attacks can give valuable data to the attacker that can lead to big security breaches and can also take full access to the database server.

In SQL Injection attackers always try to enter a malicious SQL statement which will get executed in the database and return unwanted data to the attacker.
