Version Discloser

Version information can be used by an attacker to target a specific attack on that Version that is disclosed.

Whenever the browser sends HTTP to request to the server in response we get a response header which contains information of [Server, X-AspNet-Version, X-AspNetMvc-Version, X-Powered-By].

Server: Microsoft-IIS/10.0: The server shows information about which web server is begin used.

X-AspNet-Version shows information of which specific Asp.Net VersionUsed.

X-AspNetMvc-Version: 4.0 shows information of which ASP.NET MVC version Used. Here 4.0 is used

X-Powered-By: Asp.net shows information of which framework your website is running on.
