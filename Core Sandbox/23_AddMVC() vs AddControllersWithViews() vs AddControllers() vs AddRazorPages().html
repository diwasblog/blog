Which To Use ?

  • For Web API Project - AddControllers()

Because it doesnot have View, TempData,Tag Helpers

  • For Razor Pages - AddRazorWithPages() like it has pages 
  • Model View Controller - AddControllerWithViews() like general asp.net MVC project
  • AddMVC() - Has All features of above 3. It has extra features which maynot use always and will decrease performance of application.
