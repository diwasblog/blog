In both synchronous and asynchronous, pattern of execution is always from top to bottom.

Synchronous

let a = 1; let b = 2; console.log("Synchronous"); console.log(a); console.log(b);

ASynchronous

as setTimeout is async so, code below setTimeout will not wait for 1 sec and will execute and then after 1sec "Async" is displayed let a = 1; let b = 2; setTimeout(function(){ console.log("Async"); },1000) console.log("Synchronous"); console.log(a); console.log(b);
