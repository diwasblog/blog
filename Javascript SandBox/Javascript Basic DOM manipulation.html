Input type Button and Submit

Then you cannot extend submitNotice() function.because as soon as you click submit then form will be posted to that action("notice/add" ) but in button we explicitly post and submit later as above.

<input type="button">

The <input type="button"> is just a button and won't do anything by itself.

       
        <form method="post" action="/notice/add"  id="noticeForm" >
        <input type="text" name="Title"   class="form-control" />
        <input type="button" value="Add" onclick="submitNotice" class="btn btn-default" />
        </form>
  
        Then you can call click event below to post
      
       function   submitNotice(){
              document.querySelector('#noticeForm').submit();
        }


        //Alternatively
        let form  = document.getElementById('noticeForm');

        form.addEventListener('submit', (event) => {
            // handle the form data
        });

<input type="submit">

The <input type="submit">, when inside a form element, will submit the form when clicked.

          
           <form method="post" action="/notice/add"    id="noticeForm" >
           <input type="text" name="Title"   class="form-control" />
           <input type="submit" value="Add" />
           </form>

Prevent Form From loading on submitting

        
            form.addEventListener('submit', (event) => {
                // ...
                // stop form submission
                event.preventDefault();
            });
