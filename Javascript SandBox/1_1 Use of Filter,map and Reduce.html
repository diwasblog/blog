Array.prototype.every

Since ES5, if you want to iterate over a portion of an array, you can use Array.prototype.every, which iterates until we return false:
// [].every() stops once it finds a false result // thus, this iteration will stop on value 7 (since 7 %2 !== 0) [2, 4, 7, 9].every(function(value, index, arr) { console.log(value); return value % 2 === 0; // iterate until an odd number is found });
