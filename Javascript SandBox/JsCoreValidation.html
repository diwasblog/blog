Validate TextField

By default all form field values are accessed as strings.So if you want numeric value you first have to convert field from strings to numbers.

var field1 = parseInt(form.field1.value); var field2 = parseInt(form.field2.value);

So if you want to add two number then you have to parse two values and then only add them.
