JSON

Do you know that the data which is exchanged between a client (web-browser) and a server can only be text? Did you remember that there is a flexible format which can convert any JavaScript object into text and send it to server, and vice versa. The solution here is JSON (JavaScript Object Notation). It is a file format for transmitting data which is built on two structures.

Example:

Suppose one person is Nepali and another Norwish and they both want to send letter.But Nepalese donot understand Norway language and vice versa. But there is one case and that is both of them can understand and write in english. So, first Nepali writes the letter in English and send it to Norwish and Norwish will understand it and then they can communicate with each other. So here english language works as JSON.

Practical : In this scenario Nepali and Norwish donot understand each other MotherTongue language , In the same way Client Side Lanaguage(JS) and Server Side Language(ASP,PHP) donot understand each other but both of them can understand Json format of data so in Client side Data are send to Json format and in server side like ASP convert that back in its Format using Serializer and then both of them are able to communicate.
