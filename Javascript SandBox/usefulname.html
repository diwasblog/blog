Useful naming convention Javascript

Bad

Good

isLoading hasInsurance isDisabled var hasEncryption = true; var isVisible = true; const isPositiveEvenNumber = isPositive && isEven; const isPositiveNumber = x > 0; const isEvenNumber = x % 2;

Useful naming convention Function (camelCase)

get, fetch, push, apply, calculate, compute, post

Bad

Good

function getName(firstName, lastName) { return `${firstName} ${lastName}`; }
