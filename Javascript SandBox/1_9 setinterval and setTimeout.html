• setTimeout: Calls a function once after a specified delay.
  • setInterval: Calls a function continuously at a specified delay between each call.

setTimeout method expects 2 arguments: a reference to a callback function and a delay in milliseconds.

Using Inline Function

<script> setTimeout(() => { console.log("loaded after 5 second"); }, 5000); </script>

Using Function

<script> function gator() { console.log('Also load after 10second!!!!'); } setTimeout(gator, 10000); </script>

WARNING!!!!Note that the function shouldn't be invoked, but instead a reference to the function should be used.

setInterval calls the callback function repeatedly with the specified delay in-between the calls.

It expects 2 arguments: a callback function and a delay in milliseconds

let i = 0; function increment() { i++; console.log(i); } setInterval(increment, 1000); //output: //1 //2 //3 //4..
