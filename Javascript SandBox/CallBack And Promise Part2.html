Callback Function

  • When a function simply accepts another function as an argument, this contained function is known as a callback function.
  • Using callback functions is a core functional programming concept.
  • A callback may or may not performed asynchronously. Normally callback runs after the parent function completes its operation.
  • Example Below: Calculate is callback function and Display() is parent function.
function calculate(value) { console.log("calculate function"); } function display(callback) { console.log("Displaying:"); callback(5); // callback === calculate and callback is defined as function so called above function } display(calculate); //Note that here: callback will be 5
