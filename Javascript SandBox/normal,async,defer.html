Normal

The first document started to parse from HTML tag, go down serially and when <script> tag is found then parsing gets stopped and js is downloaded and then executed then parsing of HTML continues again.

Async

When js is found parsing HTML will continue up to js downloaded and then js is executed (other gets stopped), then again parsing of HTML continued.

Defer

When js is found, parsing and downloading of js continue and when all HTML is parsed completely then only the javascript gets executed.

Note here: In above example we are loading js at end of head section, and defer works best for executing js after html parse, also if there are numerous js then also it load js in order manner.

So, putting js with defer tag at head section is equivalent to loading js at end of body tag because when js is put at end of body tag then all html parsing will be completed then js gets executed .

Note that if we have an image on the page then parsing HTML has no effect of the image, image download will follow own path and does no effect on parsing of html.
