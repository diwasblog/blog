Content Box

The width and height properties includes only the content. Border ,padding, or margin are not included.
<div style="background-color:gray">This is content area</div>
div{ box-sizing:content-box; width:300px; height:200px; margin:3px; border:5px solid red; padding:5px; }

========================================================================================================

Border box

The width and height includes content, padding,border, but not margin.
<div style="background-color:gray">This is content area</div>
div{ box-sizing:border-box; width:300px; height:200px; margin:3px; border:5px solid red; padding:5px; }
=================================================================================================================
