Translation (basic math)


========================================================================================================================

Transfrom in X direction

<div>Static</div> <div class="moved">Moved</div> <div>Static</div>
div { width: 60px; height: 60px; background-color: skyblue; } .moved { transform: translate(10px); /* Equal to translateX(10px) */ background-color: pink; }

====================================================================================================================

Transfrom in X and Y both direction

<div>Static</div> <div class="moved">Moved</div> <div>Static</div>
div { width: 60px; height: 60px; background-color: skyblue; } .moved { transform: translate(10px, 10px); /*translate in both X, Y direction */ background-color: pink; }

============================================================================================================================

Scale

//format scale(sx) scale(sx, sy)
<div>Normal</div> <div class="scaled">Scaled</div>
div { width: 80px; height: 80px; background-color: skyblue; } .scaled { transform: scale(0.7); /* Equal to scaleX(0.7) scaleY(0.7) */ background-color: pink; }

==========================================================================================

Scale

//format scale(sx) scale(sx, sy)
<div>Normal</div> <div class="scaled">Scaled</div>
div { width: 80px; height: 80px; background-color: skyblue; } .scaled { transform: scale(2, 0.5); /* Equal to scaleX(2) scaleY(0.5) */ transform-origin: left; background-color: pink; }
