Pseudoelement

  1. Pseudocode works only if tag has content. like <label>content</label> , <button>content</button>, <p>content</p>   but not for <input ... > , <img ... >  because it doesnot have content.
  2. We must use content:'' properties in css.

Example :

say: we have <label>content</label> then the content is added before and after the content of label but not before and after label.

Adding dynamic content before 'p' tag and after 'h2' tag <p>First</p> <h2>Diwas</h2> <p> Second</p> <style> p::before{content:"Diwas "} h2::after{content:" after "} </style>
