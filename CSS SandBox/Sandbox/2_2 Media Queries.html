Required Things for making website Responsive

Fluid Design

Make sure your website is fluid. What that means is that your website components should scale with your screen size.

For this use percentage for your height and width instead of static pixel sizes.

 This way when someone resizes their browser window, your html tags will shrink or expand accordingly instead of breaking the page.

Use Media Queries

2a Mobile First

Your small screen styles are in your regular screen CSS and then as the screen gets larger you override what you need to. So, min-width media queries in general.  

 

Always Design for Mobile First

Mobile First means designing for mobile before designing for desktop or any other device (This will make the page display faster on smaller devices).

          
              html { background: red; }

              @media (min-width: 600px) {
                html { background: green; }
              }

2 b Desktop First

Your large screen styles are in your regular screen CSS and then as the screen gets smaller you override what you need to. So, max-width media queries in general.

          
              //When the client has 600 width or less:
              html { background: red; }

              @media (max-width: 600px) {
                html { background: green; }
              }
