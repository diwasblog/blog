There are four different combinators in CSS:

  • descendant selector (space)
  • child selector (>)
  • adjacent sibling selector (+)
  • general sibling selector (~)

Descendent Selector

A descendant selector in CSS is any selector with white space between two selectors without a combinator.


Descendant means anywhere nested within it in the DOM tree.
Could be a direct child, could be five levels deep, it is still a descendant.
Example

Normal Selector

        
            div,p{
  
            }

Descendent Selector

        
          div p{
  
          }

Direct Child Selector

The child combinator (>) is placed between two CSS selectors. Elements matched by the second selector must be the immediate children of the elements matched by the first selector 
        
            <div>
            <p>Paragraph 1 in the div.</p>
            <p>Paragraph 2 in the div.</p>
            <span><p>Paragraph 3 in the div.</p></span> 
            </div>
              
            <p>Paragraph 4. Not in a div.</p>
            <p>Paragraph 5. Not in a div.</p>
        
      
        
            div > p {
              background-color: yellow;
          }

Adjacent Sibling Selector

The adjacent sibling selector selects all elements that are the adjacent siblings of a specified element.

Sibling elements must have the same parent element, and "adjacent" means "immediately following".

        
            

                <h1>Article 1</h1>
                <p>paragraph1</p> 
                <p>paragraph2</p>
                
                <h1>Article 2</h1>
                <p>paragraph3</p>   
                <p>paragraph4</p>
                
                <h1>Article 3</h1>
                <p>paragraph5</p>    
                <p>paragraph6</p>
              

        
      
        
            #allarticles h1+p{
              color:red;
            }

Pseudo Class Selector

  
      selector:pseudo-class {
        property: value;
      }
Some Pseudo class are : :active :nth-child :last-child :link :focus and soon 
        
            <p><b><a href="www.google.com" target="_blank">This is a link</a></b></p>
            <p><b>Note: a:hover MUST come after a:link and a:visited in the CSS definition in order to be effective.</p>
            <p><b>Note: a:active MUST come after a:hover in the CSS definition in order to be effective.</p>
            
        
      
        
          /* unvisited link */
          a:link {
              color: #FF0000;
          }

          /* visited link */
          a:visited {
              color: #00FF00;
          }

          /* mouse over link */
          a:hover {
              color: #FF00FF;
          }

          /* selected link */
          a:active {
              color: #0000FF;
          }
