Horizontally Centering


Text-Align Method ===> Block Level Element like div , nav,h,p

  1. enclose the div that you want to center with a parent element (commonly known as a wrapper or container).
  2. set text-align: center to the parent element.
  3. then set the inside div to display: inline-block
<div class="container"> <div class="square"></div> </div> .container { text-align: center; } .square { background-color: #0074D9; width: 100px; height: 100px; display: inline-block; }

Here parent div has container class and I want to center div with square class. Here apply text-align center to container and display inline to the square.This is because div is block-level element and takes the whole width of div. By setting the display property of my blue square to inline-block, we ensure that the blue square will only span the width that I have set, which is 100px.

Adding multiple child elements within the parent element (blue squares in this example), will center all of them.

Margin Auto Method ===> For Block Level Element like Div only

For this, we don't need a parent. We can simply apply margin: 0 auto to our yellow box, as long with a defined width.

<div class="yellow-square"></div> .yellow-square { background-color: #FFDC00; width: 100px; height: 100px; margin: 0 auto; }
