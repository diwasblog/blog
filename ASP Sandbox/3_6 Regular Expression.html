Regular Expression

Useful in extracting information from text such as code,log files, spreadsheets or even document.

Remember:

B for Bracket C for Caret D for Dollar

 

B

'[' Specify character which needs to be matched

'{' specify how many character 

'(' for grouping
C

'^'  marks starts of a pattern

^xyz matches  xyz123 (yes)

                               xyz456 (yes)

                               xyzsdfkj(Yes)

                               123xyz (No)

 
D

'$' marks end of a pattern.

pqr$ matches 123pqr   (Yes)

                                 pgr123   (No)

 

Example:

^xyz

xyzabc  (Yes)

xyz123  (Yes)

123xyz  (No)
xyz$

xyz123  (No)

123xyz   (Yes)
^xyz$

xyz   (Yes)

abc   (No)

123   (No)

abcd   (No)

Q. Enter character which exists between a-g?

           [a-g]

a Yes
b Yes
ab Yes
az yes
axy yes
z no

It only check one digit character if match then true and donot look for rest of digit.

   ie .It done for one character check.
