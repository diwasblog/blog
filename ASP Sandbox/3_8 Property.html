Auto Property(Without body)

Below is auto property (without body of property) 
  
      public class Genre
      {
          public string Name { get; set; }
      }
Internally compiler converts to below and again it gets converted to method that we describe at first 
        
            private string name;
            public string Name
            {
                get
                {
                    return this.name;
                }
                set
                {
                    this.name = value;
                }
            }
Properties are just syntactic sugar for getter and setter methods. Ie.above property gets converted as below methods and only local variable name get occupy space
