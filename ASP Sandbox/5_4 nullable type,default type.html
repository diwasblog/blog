DataTypes in C#

Data types in C# are divided into two main categories: Value Type and Reference Type. 

        
            int x=null; //error  cannot convert null to int because it is non-nullable types

But we can assign null value to integer 

          
                int ? x = null;

So we convert non nullable type to nullable types as above

Nullable types has two member : HasValue and Value

HasValue: HasValue is of type boolean value. It is set to true when the variable contains a non-null value.

          
              static void Main(string[] args)  
              {  
              int? x = 5;  
              if(x.HasValue)  
              {  
              Console.WriteLine(x.Value.ToString());  
              }  
              else  
              {  
              Console.WriteLine("contain Null value.");  
              }  
              Console.ReadLine();
        
              //Output: 5
          
        
          
              static void Main(string[] args)

Null Coalescing Operator(??) is binary operator  that simplifies checking for null values.so, It simplifies null tests.

It is used to assign a default value to a variable when the value is null.

  
      string name = null;  
      string myname = name ?? "Laxmi";  
      Console.WriteLine(myname);  //Laxmi
