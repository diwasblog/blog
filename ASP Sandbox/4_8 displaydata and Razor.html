      public class HomeController:Controller

Block String Data from Url Access

  
    [NonAction]
    public class HomeController:Controller
    {
      public string Method1()
      {
        return "Hello world";
      }
    }
Url: http://localhost:14242/home/Method1 Output: Resource cannot be accessm ============================================================== In general its a bad design to have a public method in a controller that is not an action method.If you have such method for performing business calculations, it should be somewhere in the model and not in the controller. ==========================================================================================================
