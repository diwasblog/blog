What is the .csproj file?

  • This file will have information about the files included in that project, assemblies used in that project, project GUID and project version, etc.
  • It will be automatically generated when we create.
  • One project has one csproj file. If 5 Project then 5 Csproj file associate to each project. Open each project and see the ".csproj" file
  • Multiple projects make a solution

Do we need to deploy .csproj file

The csproj file is not part of the deployed application like the web.config. It is used by the build system and run application locally. Therefore if you make changes to it, they will only take effect if you rebuild the application.
Also for deployment you have to publish your application and deploy the dlls and static pages/css images its, we should not upload whole source code to the production server.
