Var Type

It is static type and its type is known at compile time. 
    
            var x = "test";  // hover here mouse you will see x is string and is known at compile time
            int len = x.Length;   // above line is compiled so we can get Length property.  // compiled time check is possible
=======================================================================================================

Dynamic Type

It is dynamic type and its type is known at runtime 
    
            dynamic y ="test2"   // It is not known at compiled time, we only known y at runtime . Hover mouse here , it will say dynamic
            int len = y.Length;  // here we will not get intellisense for Length because y is dynamic typed, so we get automatically Length supported at run time only.
            //compile time checking isnot possible.
At compilation a dynamic is converted into a System.object and the compiler will emit the code for type safety during runtime. It suffers from boxing and unboxing because it is treated as a System.Object and also the application's performance will suffer siince the compiler emits the code for all the type safety. ============================================================================================================

Object

        
                object y ="test2"   // It is not known at compiled time, we only known y at runtime . Hover mouse here , it will say dynamic
